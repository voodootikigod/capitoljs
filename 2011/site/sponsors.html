Phono (http://phono.com/) is a simple jQuery plugin and JavaScript library that turns any web browser into a phone; capable of making and receiving phone calls. Phono can also be used to build in-browser instant messaging applications that can send and receive IM messages. You can even connect your Phono instance to SIP clients or Tropo applications for next generation telephony applications; all with a simple unified API.

Phono makes it easy to build powerful, multi-channel communication apps that run in your browser, using the jQuery and Javascript syntax you already know. Phono is available under Apache2 Software license, is available for forking on GitHub and is brought to you by the folks at Voxeo Labs.
